Lista de Descrição (dl, dt, dd)

Use dl para listas de termos e definições. dt é o termo; dd é a descrição.

Exemplo básico

HTML
Linguagem de marcação para estruturar conteúdo na web.
CSS
Camadas de estilo para apresentação visual do conteúdo.
JavaScript
Linguagem de programação para comportamento e interatividade.

Citações (blockquote, q, cite)

blockquote é usado para citações em bloco e pode ter a origem em cite. q é uma citação curta inline. O elemento cite marca o título/autor da obra.
