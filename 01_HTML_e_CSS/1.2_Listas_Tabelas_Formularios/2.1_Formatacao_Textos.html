Exemplos de formatação de texto em HTML

Negrito semântico com <strong>: Este texto é importante.

Itálico com <i>: Este texto está em itálico.

Sublinhado com <u>: Este texto está sublinhado.

Texto marcado com <mark>: Texto destacado.

Texto pequeno com <small>: Texto menor.

Texto deletado com <del>: Texto removido.

Texto inserido com <ins>: Texto adicionado.

Subscrito com <sub>: H2O
