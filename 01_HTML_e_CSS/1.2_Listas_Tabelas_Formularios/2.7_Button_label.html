Exemplo de <input> com <label> e <button>



Explicação dos elementos:

  • <label>: O elemento <label> é usado para definir um rótulo para um elemento de formulário, como <input>. O atributo for deve ser igual ao id do input correspondente, facilitando a acessibilidade e a usabilidade.
  • <input>: O elemento <input> permite ao usuário inserir dados. Existem vários tipos, como text, password, email, etc. No exemplo acima, usamos type="text" para entrada de texto.
  • <button>: O elemento <button> cria um botão clicável. Pode ser usado para enviar formulários (type="submit") ou executar ações com JavaScript (type="button").
