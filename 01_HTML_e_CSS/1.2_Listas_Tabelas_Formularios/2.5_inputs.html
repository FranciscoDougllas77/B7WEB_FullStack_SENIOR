Tipos de Inputs em HTML

O elemento <input> em HTML pode ser utilizado de diversas formas, dependendo do valor do atributo type. Abaixo estão exemplos e explicações para cada tipo de input:

1. type="text"

Campo de texto simples para entrada de uma linha.

2. type="password"

Campo para senha, oculta os caracteres digitados.

3. type="email"

Campo para e-mail, valida o formato do e-mail.

4. type="number"

Campo para números, permite apenas valores numéricos.

5. type="tel"

Campo para telefone, pode exibir teclado numérico em dispositivos móveis.

6. type="url"

Campo para URLs, valida se o valor inserido é um endereço web.

7. type="search"

Campo para buscas, pode exibir um botão de limpar em alguns navegadores.

8. type="date"

Campo para seleção de data.

9. type="time"

Campo para seleção de horário.

10. type="datetime-local"

Campo para data e hora (sem fuso horário).

11. type="month"

Campo para seleção de mês e ano.

12. type="week"

Campo para seleção de semana do ano.

13. type="color"

Campo para seleção de cor.

14. type="checkbox"

Caixa de seleção (pode marcar mais de uma opção).

15. type="radio"

Botão de opção (apenas uma opção pode ser selecionada por grupo).

16. type="file"

Campo para envio de arquivos.

17. type="range"

Controle deslizante para selecionar um valor dentro de um intervalo.

18. type="hidden"

Campo oculto, não é exibido ao usuário, mas pode enviar valores no formulário.

19. type="submit"

Botão para enviar o formulário.

20. type="reset"

Botão para limpar todos os campos do formulário.

21. type="button"

Botão genérico, pode ser usado com JavaScript.
