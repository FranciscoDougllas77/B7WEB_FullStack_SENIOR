Seletores por Atributo

Os seletores por atributo permitem selecionar elementos HTML que possuem um atributo específico ou um valor específico para esse atributo.

  • [atributo]: Seleciona elementos que possuem o atributo, independente do valor.
  • [atributo="valor"]: Seleciona elementos cujo atributo tem exatamente o valor especificado.
  • [atributo^="valor"]: Seleciona elementos cujo atributo começa com o valor especificado.
  • [atributo$="valor"]: Seleciona elementos cujo atributo termina com o valor especificado.
  • [atributo*="valor"]: Seleciona elementos cujo atributo contém o valor especificado.

Explicação:
- input[type="email"]: Aplica estilo apenas ao campo de e-mail.
- a[href^="mailto:"]: Aplica estilo aos links de e-mail.
- a[href$=".com"]: Aplica estilo aos links que terminam com ".com".
- a[href*="google"]: Aplica estilo aos links que contêm "google" no endereço.
