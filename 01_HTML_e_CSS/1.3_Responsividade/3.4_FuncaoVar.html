var(), calc(), min(), max()

Exemplos práticos de funções CSS para responsividade e design tokens.

Use variáveis de tema e valores alternativos quando ausentes.

calc() com var()

Calcula largura interna subtraindo paddings e bordas.

width: calc(100% - (var(--space) * 2) - 2px)

min()

Limita o crescimento para evitar linhas muito longas.

width: min(100%, 480px)

max()

Garante um tamanho mínimo confortável em telas estreitas.

width: max(320px, 40%)

Combinar min() + max()

Controla faixa de crescimento: cresce com a viewport, com mínimo e máximo.

width: min(90vw, max(360px, 720px))
