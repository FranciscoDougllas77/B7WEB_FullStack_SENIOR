Exemplo de Margin

Margin personalizada:
margin: 40px 20px 10px 60px;
(topo, direita, baixo, esquerda)

Exemplo de Padding

Padding personalizado:
padding: 40px 20px 10px 60px;
(topo, direita, baixo, esquerda)

Comparação Visual

Margin grande, sem padding
Sem margin, padding grande

Dica: Use margin para afastar elementos entre si e padding para afastar o conteúdo da borda do elemento.
