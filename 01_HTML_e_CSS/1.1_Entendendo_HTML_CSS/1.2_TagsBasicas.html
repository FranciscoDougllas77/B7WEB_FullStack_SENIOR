Tags Básicas do HTML5

O HTML5 possui diversas tags básicas para estruturar o conteúdo de uma página.

  • <h1> a <h6> - Títulos de diferentes níveis
  • <p> - Parágrafo de texto
  • <br> - Quebra de linha
  • <hr> - Linha horizontal
  • <strong> - Texto em negrito
  • <em> - Texto em itálico
  • <ul>, <ol>, <li> - Listas
  • <img> - Imagem
  • <a> - Link
  • <div> - Divisão de bloco
  • <span> - Elemento em linha

Visite o MDN Web Docs para saber mais sobre as tags HTML.
